Example of Alchemy that can easily query a locally running neo4j database.

Querying 127.0.0.1:7474


Enter search arguments seperated by a comma

The "query label" setting takes just one argument, a node label.

The cypher template looks like: "MATCH (n:{0})-[r]-(nn) RETURN n, nn, r LIMIT 100"


The "query by property" setting takes two arguments. Ex: "name, Melissa"

The cypher template looks like: "MATCH (n)-[r]-(nn) WHERE n.{0} = '{1}' RETURN n, nn, r LIMIT 100"


Custom queries can be easily added
